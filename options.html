Chrome Extension Options & Help

How It Works

This extension helps you manage your time on distracting websites by automatically closing tabs after a configurable time limit.

  • Set custom time limits for specific websites (1-120 minutes)
  • Automatic tab closure when time expires
  • Real-time timer tracking in the popup
  • Badge counter showing active timers
  • Easy configuration through the popup interface

Usage Instructions

  1. Add a website: Click the extension icon and enter a website (e.g., "youtube.com") with a time limit in minutes.
  2. Browse normally: When you visit a configured website, a timer starts automatically.
  3. Timer expires: After the set time, if you're still on the site, you'll be redirected to a quick focus plan page.
  4. Manage sites: Use the popup to edit time limits or remove sites from monitoring.

Default Sites

The extension comes pre-configured with common distracting websites and suggested time limits:

  • YouTube Shorts: 1 minutes
  • YouTube: 5 minutes
  • Facebook: 2 minutes
  • Twitter: 3 minutes
  • Instagram: 2 minutes
  • TikTok: 2 minutes
  • Reddit: 10 minutes
